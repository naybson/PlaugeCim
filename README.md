# ☣️ PlagueCim — Terminal Disease Simulation in C

PlagueCim is a **terminal-based disease simulation game written in C**, inspired by *Plague Inc*.

The player creates a disease, customizes its parameters, chooses where the outbreak begins on a world map, and watches the infection spread in real time.  
During the simulation, the disease evolves through symptoms and mutations, while humanity reacts by increasing awareness and developing a cure.

At the end of the run, the program displays a final summary and simulation report.

> ⚠️ **Windows-oriented project**  
This project relies on console behavior and rendering designed primarily for Windows terminals / Visual Studio.

---

## ✨ Features

✅ Interactive setup menu for disease configuration  
✅ Choose disease name and outbreak starting point  
✅ Select **Patient Zero** directly on the world map  
✅ Real-time disease spread simulation  
✅ Symptom system with mutations and progression tree  
✅ Cure and awareness system reacting to infection/death levels  
✅ Ports, ships, and sea-route spreading mechanics  
✅ End-of-simulation report and graph view  
✅ Fully written in **C**
