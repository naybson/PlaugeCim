# 🦠 PlagueCim — ASCII Plague Simulation

A C-based epidemic simulator that models the spread of a plague across an ASCII world map.  
Tune parameters, release Patient Zero, and watch the world fight for survival — one terminal cell at a time.

This is a sandbox simulation project to explore:
- Infection dynamics  
- Regional resistance (hot/cold, urban/rural)  
- Mutation effects  
- Travel-based transmission (ships cross oceans!)  
- Awareness & cure development over time  
- Randomness with seeds for repeatability  

---

## ✨ Features

| Feature | Description |
|--------|-------------|
🌍 Dynamic ASCII world | Color-mapped population & infection states  
🚢 Ship-based transmission | Boats act as global spread vectors  
🧬 Mutation system | Random disease evolution built in  
🏙️ Environment effects | Cities vs rural / cold vs hot zones  
💉 Cure mechanics | Public awareness & cure progress  
📊 End-screen analytics | Graphs + event markers  
🔁 Save & replay diseases | Run same strain with new conditions  
🎛️ Configurable via menus | Tweak everything live
